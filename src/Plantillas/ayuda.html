CONSIDERACIONES

1) No ingresar caracteres especiales, la letra Ñ (ñ), ni caracteres con tíldes.

2) El formato para fechas es AAAA-MM-DD, por ejemplo 1998-03-14.

3) El sexo es el caracter 'F' para femenino y 'M' para masculino.

4) Solo se aceptaran numeros de celular de hasta 12 digitos.

5) Solo se aceptan contraseñas de 8 a 10 caracteres, minimo una minuscula, mayuscula, numero y caracter especial.

6) Todos los campos que contengan espacios entre medio, deben ser reemplazados por el barra baja _ EJ: Agustín_Rossi

Casos de uso Sistema para EMAIL

CU1: GESTIONAR PERIODO
Registrar Periodo
FORMATO --> REGPER: nombre:semestre:año
EJEMPLO --> REGPER: 2022-2:2:2022
Modificar datos de un periodo
FORMATO --> EDITPER: id:nombre:semestre:año
EJEMPLO --> EDITPER: 3:2022-1:1:2022
Eliminar un Periodo
FORMATO --> DELPER:id
EJEMPLO --> DELPER:4
Ver datos del periodo
FORMATO --> VERPER:id
EJEMPLO --> VERPER:3
Listar periodos
FORMATO --> LISPER:fechaInicio:fechaFin
EJEMPLO --> LISPER:2022-01-01:2022-12-30
CU8: GESTIONAR REPORTES
Reporte postulante por periodo
FORMATO --> REPORPOSPER:fechaInicio:fechaFin
EJEMPLO1 --> REPORPOSPER:2022-01-01:2022-03-10
Reporte Auxiliar por carga horaria
FORMATO --> REPORAUXCAR:todos
EJEMPLO --> REPORAUXCAR:*
Reporte Materia por producto
FORMATO --> REPORMATGRUP
EJEMPLO --> REPORMATGRUP:*
CU4: GESTIONAR MATERIAS
Listar Materias
FORMATO SIN FILTROS --> LISMAT
FORMATO CON FILTROS --> LISMAT:nombre=valor:sigla=valor
EJEMPLO SIN FILTROS --> LISMAT (devuelve todas)
EJEMPLO --> LISMAT:nombre=Discretas:sigla=INF (devuelve filtrado)
Registrar Materia
FORMATO --> REGMAT:nombre=valor:sigla=valor
EJEMPLO --> REGMAT:nombre=Estructuras_Discretas:sigla=INF119
Ver una Materia
FORMATO --> VERMAT:id=valor
EJEMPLO --> VERMAT:id=12
Eliminar una Materia
FORMATO --> DELMAT:id=valor
EJEMPLO --> DELMAT:id=15
CU4: GESTIONAR GRUPOS DE MATERIA
Listar Grupos
FORMATO SIN FLITROS --> LISTMATGRUP
FORMATO CON FILTROS --> LISTMATGRUP:materia=valor:sigla=valor:grupo=valor:carrera=valor
EJEMPLO SIN FILTROS --> LISTMATGRUP
EJEMPLO CON FILTROS --> LISTMATGRUP:materia=Prog:sigla=INF:grupo=SB:carrera=sistemas
Registrar Grupo
FORMATO --> REGMATGRUP:materia=valor:grupo=valor:horario=valor:docente=valor:carrera=valor
EJEMPLO --> REGMATGRUP:materia=INF119:grupo=SA:horario=19.45-21.30_lun-mie-vi:docente=3:carrera=2
CU5: GESTIONAR POSTULANTES
Listar Postulantes
FORMATO SIN FILTROS --> LISPOS
FORMATO CON FILTROS --> LISPOS:nombre=valor:periodo=valor:carrera=valor
EJEMPLO SIN FILTROS --> LISPOS
EJEMPLO --> LISPOS:nombre=Javier:periodo=2:carrera=2
Registrar Postulante
FORMATO --> REGPOS:nombres=valor:apellidos=valor:registro=valor:ci=valor:carrera=valor:periodo=valor:materias=valor1,valor2...
EJEMPLO --> REGPOS:nombres=Rodrigo_Jesus:apellidos=Duarte_Barrientos:registro=218017243:ci=8616461:carrera=1:periodo=1:materias=INF110,INF119,INF220
Ver Postulante
FORMATO --> VERPOS:registro=valor
EJEMPLO --> VERPOS:registro=218017243
CU6: GESTIONAR EVALUACIONES
Listar Evaluaciones
FORMATO SIN FILTROS --> LISEVAL
FORMATO CON FILTROS --> LISEVAL:sigla=valor:periodo=valor:docente=valor
EJEMPLO SIN FILTROS --> LISEVAL
EJEMPLO --> LISEVAL:sigla=INF110:periodo=1-2021:docente=Jorge_Gonzalez
Listar Evaluaciones de Postulante
FORMATO --> LISEVALPOS:registro=valor
EJEMPLO --> LISEVALPOS:registro=218017243
Registrar Evaluacion
FORMATO --> REGEVAL:materia=valor:periodo=valor:docente=valor:fecha=valor
EJEMPLO --> REGEVAL:materia=INF115:periodo=3:docente=4:fecha=2022.08.19-19.45
